BAIXAR APK (Android)

Instruções de Instalação

  1. Clique no botão **BAIXAR APK** acima.
  2. Se o navegador alertar sobre "arquivo perigoso", selecione **Manter/Permitir** (é seguro, pois está vindo do seu servidor).
  3. Após o download, toque no arquivo `.apk` e selecione **Instalar**.
  4. Se o Android bloquear a instalação, vá para Configurações e permita a instalação de "fontes desconhecidas" para o seu navegador.
